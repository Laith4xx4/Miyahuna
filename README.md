# Miyahuna (ميّاهنا) - Jordan Location Water Schedule

منظومة ذكية للتعرف على المناطق والجداول الزمنية لأدوار المياه في الأردن باستخدام البحث التقريبي والتوحيد التلقائي للغة العربية.

## الميزات
- **محرك مناطق ذكي**: يحتوي على قاعدة بيانات شاملة لكافة محافظات ومناطق الأردن.
- **البحث التقريبي (Fuzzy Search)**: يدعم التعرف على الكلمات حتى لو كانت تحتوي على أخطاء إملائية.
- **توحيد النصوص العربية (Arabic Normalization)**: إزالة التشكيل، توحيد الألف والياء والتاء المربوطة.
- **تحديد ذكي للأوقات والمناطق**: يستخرج المناطق والأوقات (صباحاً، ظهراً، مساءً) تلقائياً من الإعلانات الخام.
